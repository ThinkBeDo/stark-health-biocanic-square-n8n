# Stark Health: Biocanic to Square n8n Workflow

This repository contains the project pillar document and supporting documentation for the Stark Health integration workflow connecting Biocanic EHR to Square payment processing via n8n automation.

## Purpose

Centralized documentation repository for AI-assisted project management and development, enabling both Claude and ChatGPT to read and edit project documentation through their GitHub MCP connectors.

## Repository Structure

- `project_pillar.md` - Main working document (single source of truth)
- `notes/` - Supporting documentation and version snapshots
- `README.md` - This file

## Access Setup

### For ChatGPT
Settings → Connected apps → GitHub → Enable "Read and write"

### For Claude  
Project → Add Knowledge Source → GitHub Repo → Select this repo

## Workflow

1. AI assistants can directly read, edit, and commit changes to markdown files
2. All changes are tracked via GitHub commit history
3. Human oversight through GitHub web interface or local git client
4. Version control enables rollback if needed
